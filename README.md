# Topological Syntax in Killer Whale Communication

A reaction-diffusion topological framework for analysing cetacean vocalisations, applied to ~40,000 killer whale calls across four ecotypes and 18 years of recordings (DCLDE 2027 dataset, 2005-2023).

**Paper:** [paper/PAPER.pdf](paper/PAPER.pdf) | **Executive summary:** [paper/EXECUTIVE_SUMMARY.pdf](paper/EXECUTIVE_SUMMARY.pdf)

## Headline findings

| Finding | Result | Significance |
|---|---|---|
| **Topological syntax** | acoustic similarity predicts sequential order, Cohen's d = 1.34-1.57 | grammar generated by signal geometry |
| **Boltzmann grammar** | transition probability = exp(similarity / T), R² > 0.96 | a single temperature parameter fits the data |
| **Unprecedented sequential memory** | MI half-life 45.4 calls (TKW) | 6-15× longer than any other non-human species |
| **Higher-order syntax** | Markov order >4 | in the human language range (5-7) |
| **Ecological stress detection** | p = 10⁻²⁹⁶ | 2019 prey crisis shifted SRKW communication |
| **Population-specificity** | 20× JSD vs SAR/TKW control | natural experiment rules out generic artefacts |
| **Day/night grammar switch** | distinct regimes within one population | code-switching, not noise |
| **Linguistic universals** | Zipf, Heaps, brevity | orca calls obey the same laws as human language |

Full methodology and 58 findings: [FINDINGS.md](FINDINGS.md).

## The core claim

Orca communication is a **topological language** — acoustic proximity generates sequential structure directly. This is fundamentally different from human language's categorical syntax, where a word's sound does not predict what comes next. In orca communication, it does.

```
Human:  vocabulary + compression + abstraction
Orca:   memory + coherence + topological coupling
```

Neither is "more advanced." Orthogonal innovations.

## Quick start

```bash
git clone https://github.com/cmiklov/orca-paper
cd orca-paper
pip install -r requirements.txt

# Run the test suite (synthetic data only — no external downloads)
python -m pytest tests/ -v

# Reproduce a finding from the included extracted features
python analyse_topology.py     # topological syntax (the crown jewel)
python analyse_boltzmann.py    # grammar temperature
python analyse_universals.py   # Zipf, Heaps, brevity

# To extract features from raw audio, fetch the Orcasound exemplars first
git clone https://github.com/orcasound/signals-srkw data/signals-srkw
python extract_real_features.py
```

The extracted feature `.npz` files under [`data/`](data/) are committed so the analysis scripts can run without re-extracting from raw audio. Total ~11 MB.

## Methodology in brief

The analysis treats orca call sequences as the output of a **reaction-diffusion (R/D)** dynamic system, where:

- **Reaction** — calls modify the immediate acoustic neighbourhood (the call you just made constrains what comes next)
- **Diffusion** — context spreads across the population over time (dialects, drift, stress responses)

Two semantic fields run in parallel:
- **Acoustic topology** — what calls sound like (50-D feature space: spectral, temporal, harmonic)
- **Context topology** — when/where calls are used (behavioural, station, ecotype)

When these two fields agree on the geometry, you have a stable language. When they diverge — under ecological stress, for example — the grammar shifts. The 2019 SRKW prey crisis appears as a measurable change in transition probabilities (p = 10⁻²⁹⁶) before any vocalisation count anomaly.

Cross-population dialects are recovered as **Procrustes rotations** between the two ecotypes' field embeddings. Topology rotation is translation.

Paper is pre-submission. Four pre-submission vulnerabilities (Boltzmann permutation null, 20 ms ICI artefact, acoustic priming alternative, Markov order CIs) were identified in internal review and addressed in [`paper/REVIEW_RESPONSE.md`](paper/REVIEW_RESPONSE.md) and the `vuln*.py` scripts.

Findings, methodology, and code are released openly so they can be replicated, critiqued, and extended. Issues and pull requests welcome.
